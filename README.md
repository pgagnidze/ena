## მაგალითი

```ena
ა = 20;
ბ = 100;
თუ ბ != ა {
  ბ = 1
} თუარადა ბ > ა {
  ბ = 2
} თუარა {
  ბ = 3
};
მჩ ბ;
დააბრუნე ბ;
```
